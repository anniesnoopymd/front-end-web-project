      歡迎加入綠的國際企業股份有限公司

      ●公司簡介

      綠的傢俱成立於民國73年，因專業管理門市與相關業務的推動，並秉持有信譽的品牌持續帶給消費者信心，
      更首創一家購買、全省服務，擁有完整的倉儲運送系統，提供客戶完整的售前及售後服務。
      本公司擁有優秀的經營團隊，秉持著『熱忱、務實、專業』的經營理念，追求企業永續經營及成長；
      除整體營運穩定外，獲利狀況也逐年提昇。我們重視每一位員工，除了有良好工作環境、
      也提供學習及成長的空間，熱忱歡迎優秀的朋友一起加入綠的傢俱的工作行列。

      ●主要商品 / 服務項目

      人性化的系統傢俱，一家購買全省服務。
      室內裝潢設計、規劃、施工。
      辦公室規劃設計、施工、傢俱販售。
      居家傢俱、家飾相關產品販售。

      ●福利制度

      獎金 / 禮品類 1.年終獎金
      保險類 : 1.勞保 2.健保 3.員工團保 4.意外險 5.職災保險
      休閒類 : 1.國內旅遊 2.國外旅遊
      制度類 : 1.員工制服 2.伙食費 3.績效獎金 4.完整的教育訓練 5.順暢的升遷管道
      請 / 休假制度: 1.特休/年假 2.旅遊假 3.陪產假
      其他 : 1.員工購物優惠
      補助類 : 1.結婚禮金 2.生育津貼 3.員工進修補助 4.旅遊補助 5.住院慰問金 6.退職金提撥

      ●經營理念

      公司的經營理念「服務親切專業、顧客滿意喜悅、產品獨特優質、價格實在誠實」。更秉持務實、專業、熱忱的企業文化，實行開放式的管理體制。

      ●薪獎制度

      門市店長：管理職（底薪+獎金，年收入可達60萬以上） 獎金 (依店績效核算)
      門市銷售人員 底薪, 獎金 (依個人績效核算，做越多領越多)

      ●培訓規劃

      新進儲備幹部(店長)
      新人訓：各項產品知識、專業技巧、銷售話術等 專業訓：設計技巧與運用 店長訓：領導與管理、店面管理技巧、報表運用技巧、市場分析、危機處理等
      門市銷售人員
      新人訓：各項產品知識、專業技巧、銷售話術等 專業訓：設計技巧與運用 每季外聘專業講師授課
