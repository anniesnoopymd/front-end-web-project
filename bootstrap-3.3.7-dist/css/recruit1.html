      綠的國際企業股份有限公司



      民國73年，創辦人林明清董事長，吳益財總經理以及幾位好朋友共同創立軒茂木業股份有限公司。當時以經營批發業務為主，產品項目有電腦桌、學生書桌、書櫃、電視櫃等產品。爾後由於批發市場漸漸式微，為了長期發展，公司開始調整經營方向，決定轉型經營傢俱品牌通路，直接面對消費者，並於民國88年更改公司名稱為綠的國際企業股份有限公司。

      二十多年來深根經營傢俱通路, 始終秉持「務實、專業、熱忱」的公司經營理念，堅持提供消費者「環保、時尚、安心」的高品質家具。 旗下「綠的傢俱」品牌經過二十多年努力， 迄今榮獲國內外多項品牌大獎的肯定及優良品質認證， 已是國內知名品牌。

      為了滿足消費者更多層次品味的居家家具需求，於民國100年自創品牌 inmodi。冀望為台灣的消費者帶來更具有深度、有特色、有品味且更具美感的當代生活。

      使命

      「綠的是顧客最信賴的居家顧問」是我們的使命，我們整合旗下理想家室內裝修公司的裝潢/室內設計能力，為顧客提供更完整的居家規劃服務，健康環保是我們一貫的堅持！

      願景

      「綠的國際」努力成為系統家具的卓越領導品牌，唯有持續不斷的進步成長，才能滿足顧客愈來愈多的需求。今後我們將為您我優質的品味生活繼續努力！
