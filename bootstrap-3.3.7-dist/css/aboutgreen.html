      品味生活旅程就從綠的傢俱開始吧 !
      Green is life !

      「綠的國際企業股份有限公司」於民國83年為配合世界自然生態保護觀念, 成立《綠的傢俱》全省連鎖店. 產品符合歐洲標準的環保規範, 秉持以歐洲進口的「無毒綠建材」為客戶打造安全舒適的空間. 並擁有自營工廠及完整的生產運送系統, 能確保產品運輸及客戶服務的高品質.

      門市服務人員專業親切, 累積多年的豐富經驗與創新設計概念, 並提供綠的傢俱客戶專屬「系統設計師」, 專業的設計師團隊們能客製化地整體規劃, 為每位客戶量身打造全新面貌的居家品味. 此外, 綠的傢俱更創新提出「攜帶式傢俱」的新生活概念以及「節省荷包同時享受高品質」的新生活美學, 換屋時無負擔, 讓您原有的系統傢俱在新空間能以嶄新面貌重新亮相. 這才是環保生活學的精髓 !

      隨著國人生活品質的逐漸提升, 我們除了提供客戶規格化的產品之外, 也不斷地自我求新求變, 藉由創新的客製化設計與精湛的工藝製程, 為每一位客戶量身打造獨特品味的環保居家風格, 讓所有人都能擁有與歐美同步的質感及時尚生活品味. 以客為尊, 讓客戶安心、放心、是我們每一位團隊成員的服務使命, 我們不僅提供客戶最完善的售前、售後服務, 同時也提供客戶信賴安心的永久保修承諾, 就如同傢俱是您朝夕相處的親密夥伴,「綠的傢俱」也以成為您一輩子的朋友自許.

      猶如「綠的傢俱」品牌承諾：環保、時尚、安心. 我們將積極持續、專業努力地拓展傢俱通路市場.
