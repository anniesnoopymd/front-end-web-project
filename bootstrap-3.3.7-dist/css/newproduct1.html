      輕裝修整合系統規劃打造環保空間 – 綠的傢俱 / 理想家室內裝修

      2017/01/20

      經歷過裝修經驗屋主們，許多人都曾經因為工程一再延宕，無法按照預定計劃入住而感到困擾，其實只要選擇輕裝修搭配環保系統傢俱，透過整合規劃，輕鬆即可達成縮短工程及節省預算花費。裝潢最花錢地方便在木作設計，計費方式較不透明，以後搬家又帶不走，施作時還可能產生甲醛、過敏原等有害物質。因此，歐美先進國家的空間設計都採取減少木作，多利用系統傢俱、軟飾配件搭配，營造時尚健康兼具的生活空間。

      這次很開心前往 綠的傢俱 / 理想家室內裝修 位於八里的展示間，訪問到有 25 年設計經驗的總監，透過實際居家展示，如何巧妙結合輕裝修跟系統傢俱整合運用，讓屋主們更能實際體驗整體氛圍，也很推薦大家如果有裝修需求，先找尋專業門市洽詢。

      善用輕裝修 + 系統傢俱的「整合規劃」

      空間裝潢事前準備動作，第一步建議大家同時將裝修與系統傢俱合併思考，除了工程時間掌控外，同時可創造屋主專屬健康居家風格。以綠的傢俱 / 理想家室內裝修來說，向來貼心站在使用者需求面思考，而現在許多室內設計案，擔心裝潢過程引起過敏原問題，希望多以系統傢俱配置為主，打造無毒環保生活。總監表示，特別是兒童房，整體規劃時可預留牆面，利用系統傢俱取代，再搭配理想家室內裝修的設計師量身定製，擁有更為一體性風格，達到裝潢跟系統傢俱無縫接軌，展現高度整合效果，讓入住後使用時順手便利。從大處著眼小處著手的思維模式，並讓綠的傢俱 / 理想家室內裝修榮獲經濟部新創事業金質獎，成為政府認證值得信賴團隊。

      板材色彩種類選擇多創造「多元風格」

      歐美住宅居家生活多利用傢俱、家飾當擺設，少有木作裝潢，收納部分則交給系統傢俱。他們重視環保，希望從材質到空間規劃儘量別製造多餘浪費，系統傢俱有可調整/搬遷重新佈置的特性，逐漸成為歐美室內設計配置的主流。陳總監說著，風格塑造不單靠傢俱，最好同步運用系統傢俱。門板色彩、造型種類選擇多，早就跳脫以往傳統形象。像綠的傢俱每年多次到國外參訪，整合歐美風格，推出同步流行的創新設計，搭配全面性室內規劃，施作比傳統木工省去三分之一工程時間，能更快入住完美理想家。

      從裝潢開始「健康環保」

      陳總監建議，從裝潢時就應該做好把關動作，除了選擇滿足量身訂做需求，且通過綠建材標張的環保系統傢俱。傳統木作也應該儘量要求選擇環保認證的材料，維護未來數十年的居家生活健康。例如在預算許可的前提下，搭配牆壁環保塗料，可以減少木作裝潢也能增加空間的趣味感。像綠的傢俱/理想家展示間，就採用來自日本 AICA 塗料，取自大自然素材製作，利用手工塗刷呈現多變創意形態，可吸附異味、調節空氣濕度，讓空間長時間處於健康狀態。
