      理想空間





      理想家室內裝修有限公司, 是綠的國際為提供消費者最完整的整合居家設計服務, 特聘具專業證照之設計師, 成立的室內設計公司. 理想家室內裝修, 除了協助客戶進行室內改裝設計, 還整合現代家具, 系統傢俱之空間配置服務, 讓每位消費者都可以依照喜好呈現屬於自己獨特的空間設計美學.

      室內裝修與系統傢俱的結合, 除了掌握縮短施工期的未來裝修工程趨勢, 更包含以環保材質創造更佳的收納空間優勢.

      理想家室內裝修, 傳承綠的國際「務實、專業、熱忱」的精神, 延續綠的傢俱「環保、時尚、安心」的服務宗旨, 承諾消費者一個永續服務的居家設計服務, 更自許以「自然、簡約、環保」的設計概念, 帶領室內裝修產業進入一個服務系統化、價格透明化的跨時代領域.
